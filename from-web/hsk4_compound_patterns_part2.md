## 0) ตารางสรุป pattern แบบอ่านเร็ว

| Pattern | โครงสร้าง | วิธีจำ | ตัวอย่างในไฟล์นี้ |
|---|---|---|---|
| 并列式 | คำ 2 คำวางคู่กันแล้วเกิดความหมายรวม | จำเป็นคู่/จำเป็นชุด | `结果`, `经验`, `精神`, `交通` |
| 偏正式 | คำหน้าเป็นตัวขยาย คำหลังเป็นคำหลัก | ดูคำหลังว่าเป็น “ของอะไร / แบบไหน” | `教室`, `机场`, `空气`, `面包` |
| 动宾式 | กริยา + กรรม | ถามว่า “ทำอะไรกับอะไร” | `接受`, `解决`, `考试`, `练习` |
| 补充式 | คำหลังบอกผล / สภาพ / การเปลี่ยนแปลง | มักตอบว่า “ทำแล้วเกิดอะไร” | `看见`, `来不及`, `来得及` |
| 主谓式 / 状态式 | คำหน้า+คำหลังรวมเป็นสภาพ | มองเหมือน “สิ่งนี้เป็นอย่างไร” | `紧张`, `精彩`, `满意`, `免费` |

## 3) คำที่ควรสังเกตเป็นพิเศษใน Part 2

- กลุ่ม `教...` มักเกี่ยวกับการเรียน การสอน และสถานที่เรียน เช่น `教室` / `教育`
- กลุ่ม `经...` เจอบ่อยกับคำเชิงนามธรรม เช่น `经济` / `经验` / `经理` / `经过`
- กลุ่ม `结...` มักเกี่ยวกับการรวม/จบ/ผลลัพธ์ เช่น `结果` / `结婚` / `结束`
- กลุ่ม `看...` มักเป็นการรับรู้หรือความคิดเห็น เช่น `看法` / `看见`
- กลุ่ม `来...` ถ้าต่อท้ายด้วยผลหรือเวลา จะช่วยจำได้ง่าย เช่น `来不及` / `来得及`
- กลุ่ม `可...` เป็นกลุ่มคำช่วย/คำบอกภาวะที่พบบ่อยมาก เช่น `可能` / `可以` / `可是` / `可惜`
- คำที่มี `理` มักโยงกับเหตุผล ความเข้าใจ หรือการจัดการ เช่น `理解` / `理想` / `理发`
- คำที่มี `礼` มักโยงกับมารยาท ของขวัญ หรือพิธีการ เช่น `礼貌` / `礼物`
- คำที่มี `流` มักเกี่ยวกับการไหล ความต่อเนื่อง หรือความคล่อง เช่น `流泪` / `流利` / `流行`

## 4) สรุป Part 2

ในช่วงคำศัพท์หมายเลข 301-600 คำประสมที่เดาโครงสร้างได้ดีมักกระจุกอยู่ในกลุ่มนี้:

1. คำหน้า + คำหลัก แบบขยายความหมาย (`偏正式`)
2. กริยา + กรรม (`动宾式`)
3. คำที่บอกผลลัพธ์หรือการเปลี่ยนสภาพ (`补充式`)
4. คำบอกสภาพ / คุณภาพ / ภาวะ (`主谓式`)

ถ้าคุณต้องการ ฉันจะทำ **Part 3 (คำที่ 601-900)** ต่อในไฟล์เดียวกัน หรือแยกเป็นไฟล์ใหม่อีกชุดก็ได้

## 5) ตารางแยกตัวอักษรตัวอย่าง (Character-by-character)

| คำ | พินอิน | แยกตัวอักษร | ความหมายแบบจำง่าย |

| `教室` | jiàoshì | `教` = สอน + `室` = ห้อง | ห้องเรียน |

### ทริคจำเร็วจากตาราง

- ถ้าเห็น `教` มักนึกถึงการสอนหรือการศึกษา
- ถ้าเห็น `解` มักนึกถึงการแก้/ปลด/อธิบาย
- ถ้าเห็น `理` มักนึกถึงเหตุผล การจัดการ หรือความเข้าใจ
- ถ้าเห็น `礼` มักนึกถึงมารยาท ของขวัญ หรือพิธีการ
- ถ้าเห็น `流` มักนึกถึงการไหล ความต่อเนื่อง หรือความคล่อง
- ถ้าเห็น `空` มักนึกถึงอากาศหรือพื้นที่ว่าง

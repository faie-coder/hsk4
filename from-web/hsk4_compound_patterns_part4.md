## 0) ตารางสรุป pattern แบบอ่านเร็ว

| Pattern | โครงสร้าง | วิธีจำ | ตัวอย่างในไฟล์นี้ |
|---|---|---|---|
| 并列式 | คำ 2 คำวางคู่กันแล้วเกิดความหมายรวม | จำเป็นคู่/จำเป็นชุด | `相同`, `相反`, `许多`, `一样` |
| 偏正式 | คำหน้าเป็นตัวขยาย คำหลังเป็นคำหลัก | ดูคำหลังว่าเป็น "ของอะไร / แบบไหน" | `文章`, `问题`, `西瓜`, `洗手间` |
| 动宾式 | กริยา + กรรม | ถามว่า "ทำอะไรกับอะไร" | `握手`, `洗澡`, `喜欢`, `选择` |
| 补充式 | คำหลังบอกผล / สภาพ / การเปลี่ยนแปลง | มักตอบว่า "ทำแล้วเกิดอะไร" | `误会`, `无聊`, `相信`, `详细` |
| 主谓式 / 状态式 | คำหน้า+คำหลังรวมเป็นสภาพ | มองเหมือน "สิ่งนี้เป็นอย่างไร" | `相同`, `相反`, `新鲜`, `辛苦` |

## 3) คำที่ควรสังเกตเป็นพิเศษใน Part 4

- กลุ่ม `相...` มักเกี่ยวกับความสัมพันธ์ ความคล้ายคลึง หรือการมองเห็น เช่น `相同` / `相反` / `相信` / `相信`
- กลุ่ม `喜...` มักเกี่ยวกับความชอบหรือความสุข เช่น `喜欢` / `习惯`
- กลุ่ม `新...` มักเกี่ยวกับสิ่งใหม่ ข่าว หรือความสด เช่น `新闻` / `新鲜` / `心情`
- กลุ่ม `兴...` มักเกี่ยวกับความตื่นเต้น ความสนใจ หรือความสุข เช่น `兴奋` / `兴趣` / `幸福`
- กลุ่ม `信...` มักเกี่ยวกับความเชื่อ ความไว้วางใจ หรือจดหมาย เช่น `相信` / `信任` / `信心` / `信`
- กลุ่ม `学...` มักเกี่ยวกับการเรียน โรงเรียน หรือความรู้ เช่น `学习` / `学校` / `学生` / `学问`
- กลุ่ม `要...` มักเกี่ยวกับความต้องการหรือการเชิญ เช่น `要求` / `邀请` / `需要` / `要`
- กลุ่ม `有...` มักเกี่ยวกับการมี คุณลักษณะ หรือความสัมพันธ์ เช่น `有名` / `有趣` / `友好` / `友谊`
- กลุ่ม `运...` มักเกี่ยวกับการเคลื่อนไหว การเดินทาง หรือโชคชะตา เช่น `运动` / `游泳` / `运气`
- กลุ่ม `阅...` มักเกี่ยวกับการอ่านหรือการดู เช่น `阅读` / `阅历`
- กลุ่ม `正...` มักเกี่ยวกับความถูกต้อง ความปกติ หรือการดำเนินการ เช่น `正常` / `正确` / `正式` / `正在`
- กลุ่ม `知...` มักเกี่ยวกับความรู้ การรู้ หรือการตระหนัก เช่น `知道` / `知识` / `知觉`
- กลุ่ม `重...` มักเกี่ยวกับความสำคัญ ความหนัก หรือความจริงจัง เช่น `重要` / `重视` / `重点`

## 4) สรุป Part 4

ในช่วงคำศัพท์หมายเลข 901-1200 คำประสมที่เดาโครงสร้างได้ดีมักกระจุกอยู่ในกลุ่มนี้:

1. คำหน้า + คำหลัก แบบขยายความหมาย (`偏正式`) — พบบ่อยมากกับคำที่เกี่ยวกับสิ่งของ สถานที่ และคุณลักษณะ
2. กริยา + กรรม (`动宾式`) — พบบ่อยกับคำกริยาที่บอกการกระทำในชีวิตประจำวัน
3. คำที่บอกผลลัพธ์หรือการเปลี่ยนสภาพ (`补充式`) — มักเป็นคำที่บอกอารมณ์หรือสภาพที่เกิดขึ้น
4. คำบอกสภาพ / คุณภาพ / ภาวะ (`主谓式`) — พบบ่อยกับคำคุณศัพท์ที่บอกลักษณะหรืออารมณ์

ถ้าคุณต้องการ ฉันจะทำ **Part 5 (คำที่ 1201-1500)** ต่อในไฟล์เดียวกัน หรือแยกเป็นไฟล์ใหม่อีกชุดก็ได้

## 0) ตารางสรุป pattern แบบอ่านเร็ว

| Pattern | โครงสร้าง | วิธีจำ | ตัวอย่างในไฟล์นี้ |
|---|---|---|---|
| 并列式 | คำ 2 คำที่มีน้ำหนักใกล้กัน / คู่กัน | อ่านความหมายรวม ไม่ต้องแยกตรงตัวทุกครั้ง | `东西`, `多少`, `共同` |
| 偏正式 | คำหน้าเป็นตัวขยาย คำหลังเป็นคำหลัก | ดูคำหลังว่าเป็น “ของอะไร / แบบไหน” | `电脑`, `报纸`, `方向`, `风景` |
| 动宾式 | กริยา + กรรม | ถามว่า “ทำอะไรกับอะไร” | `报名`, `参加`, `打针`, `复习` |
| 补充式 | คำหลังบอกผล / สภาพ / การเปลี่ยนแปลง | มักตอบว่า “ทำแล้วเกิดอะไร” | `出发`, `出现`, `重新` |
| 主谓式 | คำหน้าเป็นหัวข้อ คำหลังเป็นสภาพ | มองเหมือน “สิ่งนี้เป็นอย่างไร” | `高兴`, `诚实`, `成熟`, `得意` |

## 2) โครงสร้างคำประสมตาม pattern

### 2.1 并列式 — คำคู่ที่มีความหมายใกล้กัน/คู่กัน

- `东西` (dōngxi) — ของ/สิ่งของ — เดิมมาจาก “ตะวันออก + ตะวันตก” แต่ใช้เป็นคำว่า “สิ่งของ”
- `多少` (duōshao) — เท่าไหร่ — “มาก + น้อย” รวมกันเพื่อถามจำนวน
- `共同` (gòngtóng) — ร่วมกัน — “ร่วม + เหมือน/เดียวกัน”

### 2.2 偏正式 — คำหน้าเป็นตัวขยาย คำหลังเป็นคำหลัก

- `安静` (ānjìng) — เงียบสงบ — ใช้บอกสภาพ
- `安全` (ānquán) — ปลอดภัย — คำหน้าเสริมความหมายให้คำหลัง
- `爱好` (àihào) — งานอดิเรก — สิ่งที่ชอบทำ
- `爱情` (àiqíng) — ความรัก — คำรวมเชิงนามธรรม
- `办法` (bànfǎ) — วิธีการ — แนวคิด/วิธี + แบบแผน
- `报纸` (bàozhǐ) — หนังสือพิมพ์ — “ข่าว + กระดาษ”
- `标准` (biāozhǔn) — มาตรฐาน — ใช้เป็นคำคุณศัพท์/นามได้
- `部分` (bùfen) — ส่วน — เน้น “บางส่วน”
- `材料` (cáiliào) — วัสดุ — สิ่งที่ใช้เป็นองค์ประกอบ
- `方面` (fāngmiàn) — ด้าน — ใช้ระบุแง่มุม
- `方向` (fāngxiàng) — ทิศทาง — คำหน้าเสริมคำหลัง
- `风景` (fēngjǐng) — ทิวทัศน์ — “ลม/บรรยากาศ + ภาพ”
- `高级` (gāojí) — ระดับสูง — บอกระดับ
- `丰富` (fēngfù) — อุดมสมบูรณ์ — ความหมายเชิงลักษณะ
- `复杂` (fùzá) — ซับซ้อน — คำคุณศัพท์เชิงนามธรรม
- `干净` (gānjìng) — สะอาด — บอกสภาพ
- `干燥` (gānzào) — แห้ง — บอกสภาพ
- `孤单` (gūdān) — โดดเดี่ยว — บอกสภาพทางอารมณ์

### 2.3 动宾式 — กริยา + กรรม

- `安排` (ānpái) — จัดการ
- `帮忙` (bāngmáng) — ช่วยเหลือ
- `帮助` (bāngzhù) — ช่วยเหลือ
- `包括` (bāokuò) — รวมทั้ง
- `保护` (bǎohù) — ปกป้อง
- `保证` (bǎozhèng) — รับประกัน
- `报名` (bàomíng) — สมัคร/ลงชื่อ
- `参加` (cānjiā) — เข้าร่วม
- `参观` (cānguān) — เข้าชม
- `打印` (dǎyìn) — พิมพ์เอกสาร
- `打折` (dǎzhé) — ลดราคา
- `打针` (dǎzhēn) — ฉีดยา
- `道歉` (dàoqiàn) — ขอโทษ
- `调查` (diàochá) — สอบสวน/สำรวจ
- `锻炼` (duànliàn) — ออกกำลังกาย
- `访问` (fǎngwèn) — เยี่ยมเยือน
- `放弃` (fàngqì) — ยอมแพ้/ละทิ้ง
- `放心` (fàngxīn) — วางใจ
- `复习` (fùxí) — ทบทวน
- `复印` (fùyìn) — ถ่ายเอกสาร
- `负责` (fùzé) — รับผิดชอบ
- `改变` (gǎibiàn) — เปลี่ยนแปลง
- `告诉` (gàosu) — บอก
- `购物` (gòuwù) — ช้อปปิ้ง
- `鼓励` (gǔlì) — ให้กำลังใจ
- `鼓掌` (gǔzhǎng) — ปรบมือ
- `估计` (gūjì) — คาดเดา/ประเมิน
- `乘坐` (chéngzuò) — โดยสาร
- `唱歌` (chànggē) — ร้องเพลง
- `抽烟` (chōuyān) — สูบบุหรี่
- `超过` (chāoguò) — เกิน/มากกว่า
- `吃惊` (chījīng) — ตกใจ
- `迟到` (chídào) — มาสาย
- `翻译` (fānyì) — แปล
- `发展` (fāzhǎn) — พัฒนา
- `反对` (fǎnduì) — คัดค้าน
- `反映` (fǎnyìng) — สะท้อน
- `符合` (fúhé) — สอดคล้อง

### 2.4 补充式 / ผลลัพธ์หรือการเคลื่อนที่ที่ขยายความหมาย

- `出发` (chūfā) — ออกเดินทาง
- `出现` (chūxiàn) — ปรากฏตัว
- `发现` (fāxiàn) — ค้นพบ
- `发生` (fāshēng) — เกิดขึ้น
- `重新` (chóngxīn) — ใหม่อีกครั้ง

> กลุ่มนี้บางคำใกล้กับ动宾式 แต่ถ้ามองจากการใช้งานจริงจะเห็นว่า “คำหลัง” ทำหน้าที่บอกผล/สภาพที่เกิดขึ้นต่อจากการกระทำ

### 2.5 主谓式 — คำหน้าเป็นตัวหลัก คำหลังเป็นภาวะ/การบอกสภาพ

- `高兴` (gāoxìng) — ดีใจ
- `诚实` (chéngshí) — ซื่อสัตย์
- `成熟` (chéngshú) — สุกงอม/เป็นผู้ใหญ่
- `方便` (fāngbiàn) — สะดวก
- `得意` (déyì) — ภูมิใจ/พึงพอใจ
- `丰富` (fēngfù) — อุดมสมบูรณ์
- `孤单` (gūdān) — โดดเดี่ยว

## 3) คำที่ควรสังเกตเป็นพิเศษใน Part 1

- คำกลุ่ม `电...` มักเกี่ยวกับเครื่องมือหรือเทคโนโลยี เช่น `电脑` / `电视` / `电梯` / `电影` / `电子邮件`
- คำกลุ่ม `打...` มักเป็นการกระทำ เช่น `打印` / `打折` / `打针`
- คำกลุ่ม `发...` มักเป็นการเกิดขึ้นหรือการเปลี่ยนแปลง เช่น `发生` / `发现` / `发展`
- คำกลุ่ม `放...` มักมีนัยการปล่อย วาง หรือเลิกทำ เช่น `放心` / `放弃`
- คำกลุ่ม `复...` มักเป็นการทำซ้ำหรือย้อนกลับ เช่น `复习` / `复印`

## 4) สรุป Part 1

ในช่วงคำศัพท์หมายเลข 1-300 คำประสมที่เดาโครงสร้างได้ดีมักอยู่ใน 4 กลุ่มหลัก:

1. คำขยาย + คำหลัก (`偏正式`)
2. กริยา + กรรม (`动宾式`)
3. คำที่บอกผล/สภาพต่อเนื่อง (`补充式`)
4. คำบอกภาวะ/สภาพ (`主谓式`)

## 5) ตารางแยกตัวอักษรตัวอย่าง (Character-by-character)

| คำ | พินอิน | แยกตัวอักษร | ความหมายแบบจำง่าย |
|---|---|---|---|
| `阿姨` | āyí | `阿` = คำขึ้นต้นเรียกคน / `姨` = ป้าฝั่งแม่ | คำเรียกผู้หญิงวัยกลางคน |
| `爸爸` | bàba | `爸` = พ่อ + `爸` = ซ้ำเพื่อเรียก | คำเรียก “พ่อ” แบบกันเอง |
| `哥哥` | gēge | `哥` = พี่ชาย + `哥` = ซ้ำเพื่อเรียก | พี่ชาย |
| `弟弟` | dìdi | `弟` = น้องชาย + `弟` = ซ้ำเพื่อเรียก | น้องชาย |
| `儿童` | értóng | `儿` = เด็ก / ตัวอ่อน + `童` = เด็ก/เยาวชน | เด็ก/เยาวชน |
| `办公室` | bàngōngshì | `办` = จัดการ + `公` = สาธารณะ + `室` = ห้อง | ห้องสำหรับงาน/ออฟฟิศ |
| `报纸` | bàozhǐ | `报` = รายงาน/ข่าว + `纸` = กระดาษ | หนังสือพิมพ์ |
| `笔记本` | bǐjìběn | `笔` = ปากกา + `记` = บันทึก + `本` = เล่ม | สมุดจดบันทึก |
| `冰箱` | bīngxiāng | `冰` = น้ำแข็ง/เย็น + `箱` = กล่อง | กล่องเย็น = ตู้เย็น |
| `菜单` | càidān | `菜` = อาหาร/ผัก + `单` = รายการ/แผ่น | รายการอาหาร |
| `电脑` | diànnǎo | `电` = ไฟฟ้า + `脑` = สมอง | สมองไฟฟ้า = คอมพิวเตอร์ |
| `电视` | diànshì | `电` = ไฟฟ้า + `视` = ดู/มอง | ของที่ดูด้วยไฟฟ้า = โทรทัศน์ |
| `电梯` | diàntī | `电` = ไฟฟ้า + `梯` = บันได/ขั้น | บันไดไฟฟ้า = ลิฟต์ |
| `电影` | diànyǐng | `电` = ไฟฟ้า + `影` = ภาพ/เงา | ภาพที่ฉายด้วยไฟฟ้า = ภาพยนตร์ |
| `电子邮件` | diànzǐ yóujiàn | `电` = ไฟฟ้า + `子` = ชิ้นย่อย + `邮` = ไปรษณีย์ + `件` = ฉบับ/ชิ้น | อีเมล = จดหมายทางไฟฟ้า |
| `飞机` | fēijī | `飞` = บิน + `机` = เครื่องจักร | เครื่องบิน |
| `公司` | gōngsī | `公` = สาธารณะ + `司` = ดูแล/บริหาร | บริษัท/หน่วยงาน |
| `公园` | gōngyuán | `公` = สาธารณะ + `园` = สวน | สวนสาธารณะ |
| `工具` | gōngjù | `工` = งาน/ช่าง + `具` = เครื่องมือ | เครื่องมือสำหรับงาน |
| `工资` | gōngzī | `工` = งาน + `资` = ทรัพยากร/ค่าตอบแทน | เงินที่ได้จากงาน = เงินเดือน |
| `北方` | běifāng | `北` = เหนือ + `方` = ทิศ/ด้าน | ทิศเหนือ |
| `北京` | Běijīng | `北` = เหนือ + `京` = นครหลวง | ปักกิ่ง = นครหลวงทางเหนือ |
| `宾馆` | bīnguǎn | `宾` = แขก + `馆` = อาคาร/สถานที่ | โรงแรม |
| `饼干` | bǐnggān | `饼` = แผ่นขนม + `干` = แห้ง | ขนมแห้งเป็นแผ่น = บิสกิต |
| `博士` | bóshì | `博` = กว้าง/มาก + `士` = นักปราชญ์/ผู้รู้ | ผู้รู้ระดับสูง = ดุษฎีบัณฑิต |

### ทริคจำเร็วจากตาราง

- ถ้าเห็น `电` มักนึกถึงของที่เกี่ยวกับไฟฟ้า/เทคโนโลยี
- ถ้าเห็น `公` มักโยงไปทาง “สาธารณะ / ส่วนรวม / งานส่วนกลาง”
- ถ้าเห็น `工` มักโยงไปทาง “งาน / ช่าง / เครื่องมือ”
- ถ้าเห็น `报` มักโยงไปทาง “ข่าว / รายงาน / แจ้ง”
- ถ้าเห็น `笔` + `记` + `本` มักเกี่ยวกับการจดบันทึก

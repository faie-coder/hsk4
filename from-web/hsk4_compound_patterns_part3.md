> คัดจาก `hsk4-ver-01/vocab.js` ช่วงคำศัพท์หมายเลข 601-900
> อ้างอิงแนวคิดการจัดหมวดจาก `from-web/chinese_character.md`
> เลือกเฉพาะคำที่ช่วยเห็นโครงสร้างได้ค่อนข้างชัด

## 0) ตารางสรุป pattern แบบอ่านเร็ว

| Pattern | โครงสร้าง | วิธีจำ | ตัวอย่างในไฟล์นี้ |
|---|---|---|---|
| 并列式 | คำ 2 คำวางคู่กันแล้วเกิดความหมายรวม | จำเป็นคู่/จำเป็นชุด | `区别`, `全部`, `社会`, `世界` |
| 偏正式 | คำหน้าเป็นตัวขยาย คำหลังเป็นคำหลัก | ดูคำหลังว่าเป็น "ของอะไร / แบบไหน" | `面条`, `普通话`, `人民币`, `手表` |
| 动宾式 | กริยา + กรรม | ถามว่า "ทำอะไรกับอะไร" | `爬山`, `起床`, `请假`, `刷牙` |
| 补充式 | คำหลังบอกผล / สภาพ / การเปลี่ยนแปลง | มักตอบว่า "ทำแล้วเกิดอะไร" | `明白`, `难过`, `轻松`, `清楚` |
| 主谓式 / 状态式 | คำหน้า+คำหลังรวมเป็นสภาพ | มองเหมือน "สิ่งนี้เป็นอย่างไร" | `年轻`, `暖和`, `热闹`, `便宜` |

## 3) คำที่ควรสังเกตเป็นพิเศษใน Part 3

- กลุ่ม `明...` มักเกี่ยวกับความชัดเจน ความเข้าใจ หรือเวลา เช่น `明白` / `明天` / `名字`
- กลุ่ม `耐...` มักเกี่ยวกับความอดทนหรือการทน เช่น `耐心` / `难受`
- กลุ่ม `难...` มักเกี่ยวกับความยาก ความเจ็บปวด หรืออารมณ์เชิงลบ เช่น `难` / `难过` / `难受`
- กลุ่ม `年...` มักเกี่ยวกับอายุ ปี หรือเวลา เช่น `年级` / `年龄` / `年轻` / `去年`
- กลุ่ม `气...` มักเกี่ยวกับอากาศ อารมณ์ หรือสภาพแวดล้อม เช่น `气候` / `脾气` / `生气` / `天气`
- กลุ่ม `请...` มักเป็นการขอหรือการเชิญ เช่น `请` / `请假` / `请客`
- กลุ่ม `情...` มักเกี่ยวกับความรู้สึก สถานการณ์ หรือความสัมพันธ์ เช่น `情况` / `热情` / `伤心` / `心情`
- กลุ่ม `实...` มักเกี่ยวกับความจริงหรือความเป็นจริง เช่น `实际` / `实在` / `确实` / `其实`
- กลุ่ม `手...` มักเกี่ยวกับมือหรือสิ่งที่ใช้มือ เช่น `手表` / `手机` / `收拾` / `首先`
- กลุ่ม `水...` มักเกี่ยวกับน้ำหรือของเหลว เช่น `水平` / `水果` / `水` / `湿润`
- กลุ่ม `生...` มักเกี่ยวกับการเกิด ชีวิต หรือการเรียน เช่น `生命` / `生活` / `生日` / `生病`

## 4) สรุป Part 3

ในช่วงคำศัพท์หมายเลข 601-900 คำประสมที่เดาโครงสร้างได้ดีมักกระจุกอยู่ในกลุ่มนี้:

1. คำหน้า + คำหลัก แบบขยายความหมาย (`偏正式`) — พบบ่อยมากกับคำที่เกี่ยวกับสิ่งของ สถานที่ และคุณลักษณะ
2. กริยา + กรรม (`动宾式`) — พบบ่อยกับคำกริยาที่บอกการกระทำในชีวิตประจำวัน
3. คำที่บอกผลลัพธ์หรือการเปลี่ยนสภาพ (`补充式`) — มักเป็นคำที่บอกอารมณ์หรือสภาพที่เกิดขึ้น
4. คำบอกสภาพ / คุณภาพ / ภาวะ (`主谓式`) — พบบ่อยกับคำคุณศัพท์ที่บอกลักษณะหรืออารมณ์

ถ้าคุณต้องการ ฉันจะทำ **Part 4 (คำที่ 901-1200)** ต่อในไฟล์เดียวกัน หรือแยกเป็นไฟล์ใหม่อีกชุดก็ได้

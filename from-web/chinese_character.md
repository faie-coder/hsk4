# หมวดหมู่คำประสมภาษาจีน

> คู่มือจัดหมวดหมู่คำประสมภาษาจีนตาม pattern และโครงสร้าง

---

## ส่วนที่ 1: หมวดหมู่พื้นฐานสำหรับชีวิตประจำวัน

### 1. หมวดร่างกาย (Body Parts)
มักใช้โครงสร้าง "ระบุส่วน + คำหลัก"

- 头发 (tóufa): เส้นผม (头 หัว + 发 ผม)
- 眼睛 (yǎnjing): ดวงตา (眼 ตา + 睛 ลูกตา)
- 耳朵 (ěrduo): หู (耳 หู + 朵 ลักษณนามของดอกไม้/หู)
- 手腕 (shǒuwàn): ข้อมือ (手 มือ + 腕 ข้อต่อ)

### 2. หมวดอาหารและเครื่องดื่ม (Food & Drinks)
มักใช้โครงสร้าง "ขยาย + หลัก" หรือ "กริยา + กรรม"

- 面条 (miàntiáo): บะหมี่ (面 แป้ง + 条 เส้น)
- 牛奶 (niúnǎi): นมวัว (牛 วัว + 奶 นม)
- 炒饭 (chǎofàn): ข้าวผัด (炒 ผัด + 饭 ข้าว)
- 果汁 (guǒzhī): น้ำผลไม้ (果 ผลไม้ + 汁 น้ำ/สกัด)

### 3. หมวดสิ่งของเครื่องใช้ (Common Objects)
มักบอกหน้าที่หรือลักษณะของสิ่งนั้น

- 手机 (shǒujī): โทรศัพท์มือถือ (手 มือ + 机 เครื่องจักร)
- 电脑 (diànnǎo): คอมพิวเตอร์ (电 ไฟฟ้า + 脑 สมอง) — เปรียบเหมือนสมองไฟฟ้า
- 洗衣机 (xǐyījī): เครื่องซักผ้า (洗 ซัก + 衣 เสื้อผ้า + 机 เครื่อง)
- 眼镜 (yǎnjìng): แว่นตา (眼 ตา + 镜 กระจก/เลนส์)

### 4. หมวดสถานที่ (Places)

- 学校 (xuéxiào): โรงเรียน (学 เรียน + 校 สถานศึกษา)
- 商店 (shāngdiàn): ร้านค้า (商 การค้า + 店 ร้าน)
- 洗手间 (xǐshǒujiān): ห้องน้ำ (洗 ซักล้าง + 手 มือ + 间 ห้อง)

**ทริคการจำ:** ถ้าคุณรู้ความหมายของตัวอักษรจีนทีละตัว (Character) คุณจะเดาความหมายของคำประสมใหม่ๆ ได้ง่ายขึ้น

## ส่วนที่ 2: โครงสร้างคำประสม (Compound Word Structures)

### 1. แบบขนาน (并列式 - bìngliè shì)
เป็นการนำคำที่มีความหมายใกล้เคียงกัน ตรงข้ามกัน หรือประเภทเดียวกันมาวางคู่กัน

- ความหมายเหมือน/ใกล้เคียงกัน: เช่น 朋友 (péngyǒu) - เพื่อน (朋 = เพื่อน + 友 = เพื่อน)
- ความหมายตรงข้ามกัน: เช่น 大小 (dàxiǎo) - ขนาด (大 = ใหญ่ + 小 = เล็ก)
- ความหมายใหม่: เช่น 矛盾 (máodùn) - ความขัดแย้ง (矛 = หอก + 盾 = โล่)

### 2. แบบขยาย (偏正式 - piānzhèng shì)
คำหน้าทำหน้าที่ขยายคำหลัง เพื่อระบุลักษณะหรือประเภทให้ชัดเจนขึ้น

- 大米 (dàmǐ) - ข้าวสาร (大 = ใหญ่ + 米 = ข้าว)
- 黑板 (hēibǎn) - กระดานดำ (黑 = ดำ + 板 = แผ่น/กระดาน)

### 3. แบบกริยา-กรรม (动宾式 - dòngbīn shì)
คำหน้าเป็นคำกริยา และคำหลังเป็นกรรมที่ถูกกระทำ

- 吃饭 (chīfàn) - กินข้าว (吃 = กิน + 饭 = ข้าว)
- 开车 (kāichē) - ขับรถ (开 = เปิด/ขับ + 车 = รถ)

### 4. แบบกริยา-ผลลัพธ์ (补充式 - bǔchōng shì)
คำหลังทำหน้าที่เสริมหรือบอกผลลัพธ์ของการกระทำในคำแรก

- 看见 (kànjiàn) - มองเห็น (看 = มอง + 见 = พบ/เห็น)
- 改善 (gǎishàn) - ปรับปรุงให้ดีขึ้น (改 = แก้ไข + 善 = ดี)

### 5. แบบประธาน-กริยา (主谓式 - zhǔwèi shì)
คำหน้าเป็นประธานหรือตัวหลัก และคำหลังเป็นภาคแสดงหรือกริยา

- 地震 (dìzhèn) - แผ่นดินไหว (地 = ดิน/พื้น + 震 = สั่นสะเทือน)
- 年轻 (niánqīng) - อายุยังน้อย (年 = ปี/อายุ + 轻 = เบา/น้อย)

**ข้อควรรู้เพิ่มเติม:** ในการเรียนพินอิน (Pinyin) อาจจะเจอคำว่า "สระประสม" (Compound Vowels) ซึ่งเป็นการผสมเสียงสระ 2-3 เสียงเข้าด้วยกัน เช่น ai, ao, ou, ia เป็นต้น ซึ่งต่างจากคำประสมที่เป็นเรื่องของความหมายคำศัพท์

## ส่วนที่ 3: หมวดหมู่เพิ่มเติม

### 5. หมวดความรู้สึกและจิตใจ (Feelings & Thoughts)
ในภาษาจีน คำประสมที่เกี่ยวกับความรู้สึกมักมีคำว่า 心 (xīn - หัวใจ) เป็นส่วนประกอบหลัก เพราะคนจีนโบราณเชื่อว่าหัวใจเป็นที่เกิดของความคิดและความรู้สึก

- 小心 (xiǎoxīn): ระมัดระวัง (小 เล็ก + 心 ใจ)
- 放心 (fàngxīn): วางใจ/หมดห่วง (放 วาง/ปล่อย + 心 ใจ)
- 担心 (dānxīn): เป็นห่วง/กังวล (担 แบก + 心 ใจ)
- 开心 (kāixīn): ดีใจ/มีความสุข (开 เปิด + 心 ใจ)

### 6. หมวดธรรมชาติและลมฟ้าอากาศ (Nature & Weather)
มักใช้โครงสร้างแบบขยาย โดยใช้คำหลักที่บอกปรากฏการณ์ไว้ข้างหลัง

- 水果 (shuǐguǒ): ผลไม้ (水 น้ำ + 果 ผลลัพธ์/ผลไม้)
- 天气 (tiānqì): อากาศ (天 ฟ้า + 气 ก๊าซ/ลมปราณ)
- 太阳 (tàiyáng): พระอาทิตย์ (太 มาก/ที่สุด + 阳 พลังหยาง/ความร้อน)
- 彩虹 (cǎihóng): รุ้งกินน้ำ (彩 สีสัน + 虹 รุ้ง)

### 7. หมวดการเดินทางและขนส่ง (Travel & Transport)
หมวดนี้จะเน้นไปที่พาหนะและกิจกรรมการเดินทาง

- 火车 (huǒchē): รถไฟ (火 ไฟ + 车 รถ) — หมายถึงรถที่วิ่งด้วยฟืน/ถ่านไฟในสมัยก่อน
- 飞机 (fēijī): เครื่องบิน (飞 บิน + 机 เครื่องจักร)
- 行李 (xíngli): กระเป๋าเดินทาง/สัมภาระ (行 เดินทาง + 李 สัมภาระ)
- 机票 (jīpiào): ตั๋วเครื่องบิน (机 เครื่องบิน + 票 ตั๋ว)

### 8. หมวดการเรียนและวิชาการ (School & Study)

- 同学 (tóngxué): เพื่อนร่วมชั้น (同 ร่วมกัน + 学 เรียน)
- 语法 (yǔfǎ): ไวยากรณ์ (语 ภาษา + 法 กฎ/วิธี)
- 作业 (zuòyè): การบ้าน (作 ทำ + 业 งาน)

**จุดสังเกตเพิ่มเติม:** คำประสมในภาษาจีนจำนวนมากเป็นแบบ "คำตรงข้าม" ที่รวมกันแล้วเกิดความหมายใหม่ในเชิงนามธรรม เช่น:

- 多少 (duōshǎo): เท่าไหร่ (多 มาก + 少 น้อย)
- 大小 (dàxiǎo): ขนาด (大 ใหญ่ + 小 เล็ก)
- 高矮 (gāo'ǎi): ความสูง (高 สูง + 矮 เตี้ย)

## ส่วนที่ 4: หมวดหมู่สำหรับ HSK 4

ในการสอบ HSK 4 คำศัพท์จะเริ่มมีความซับซ้อนขึ้น โดยหลายคำไม่ได้แปลตรงตัวแบบ 1+1 = 2 แต่จะเป็นการประสมคำที่เป็นนามธรรม (Abstract) มากขึ้น

### 9. หมวดความสัมพันธ์และนามธรรม (มักเป็นคำตรงข้าม/คู่ขนาน)
คำประเภทนี้มักเกิดจากการนำคำ 2 คำมารวมกันเพื่อสร้างความหมายใหม่ที่กว้างขึ้น

- 父母 (fùmǔ): พ่อแม่ (父 พ่อ + 母 แม่)
- 左右 (zuǒyòu): ประมาณ/ราวๆ (左 ซ้าย + 右 ขวา)
- 标准 (biāozhǔn): มาตรฐาน (标 เครื่องหมาย + 准 แม่นยำ)
- 共同 (gòngtóng): ร่วมกัน (共 ร่วม + 同 เหมือน)

### 10. หมวดการทำงานและการจัดการ (พบบ่อยมากใน HSK 4)
หมวดนี้มักเป็นแบบ กริยา-กรรม หรือ ขยาย

- 打折 (dǎzhé): ลดราคา (打 ทำ + 折 หัก/ลดลง)
- 招聘 (zhāopìn): รับสมัครงาน (招 เรียก/กวัก + 聘 ว่าจ้าง)
- 奖金 (jiǎngjīn): เงินโบนัส (奖 รางวัล + 金 เงิน)
- 工资 (gōngzī): เงินเดือน (工 งาน + 资 ทรัพยากร/เงินทุน)

### 11. หมวดบุคลิกภาพและอารมณ์ (เน้นลักษณะนามธรรม)

- 性格 (xìnggé): นิสัย (性 ธรรมชาติของคน + 格 รูปแบบ)
- 幽默 (yōumò): ตลก/มีอารมณ์ขัน (ทับศัพท์จาก Humour แต่ตัวอักษร 幽 คือลึกซึ้ง + 默 คือเงียบ)
- 害羞 (hàixiū): อาย (害 รู้สึก/กลัว + 羞 อาย)
- 得意 (déyì): พึงพอใจ/ได้ใจ (得 ได้รับ + 意 ความคิด/ความต้องการ)

### 12. หมวดการเปลี่ยนแปลงและทิศทาง (แบบกริยา-ผลลัพธ์)
HSK 4 จะเน้นคำที่บอก "ผล" ของเหตุการณ์

- 随着 (suízhe): ตามที่... (随 ตาม + 着 อยู่)
- 通过 (tōngguò): ผ่าน/โดยการ... (通 ทะลุ + 过 ผ่าน)
- 既然 (jìrán): ในเมื่อ (既 เสร็จสิ้น/แล้ว + 然 เป็นเช่นนั้น)

**เทคนิคสำหรับ HSK 4: สังเกต "รากศัพท์" (Radicals)**
ถ้าคุณเจอคำประสมที่คุณไม่รู้ความหมาย ให้ดูตัวประกอบ (Component) จะช่วยเดาหมวดหมู่ได้:

- ถ้ามี 忄 (ใจ) → เกี่ยวกับความรู้สึก เช่น 愉快 (yúkuài - มีความสุข)
- ถ้ามี 讠 (พูด) → เกี่ยวกับการสื่อสาร เช่น 讨论 (tǎolùn - อภิปราย)
- ถ้ามี 钅 (โลหะ) → เกี่ยวกับเงินหรือของแข็ง เช่น 钥匙 (yàoshi - กุญแจ)

### 13. หมวดเวลาและกำหนดการ (Time & Schedule)
มักใช้โครงสร้างแบบขยาย โดยคำหลักบอกหน่วยเวลา:

- 星期 (xīngqī): สัปดาห์ (星 ดาว + 期 รอบ/กำหนด)
- 月份 (yuèfèn): เดือน (月 เดือน + 份 ส่วน)
- 分钟 (fēnzhōng): นาที (分 แบ่ง + 钟 นาฬิกา)
- 刚才 (gāngcái): เมื่อกี้ (刚 เพิ่ง + 才 เมื่อสักครู่)
- 以后 (yǐhòu): หลังจากนี้ (以 จาก + 后 หลัง)
- 以前 (yǐqián): ก่อนหน้านี้ (以 จาก + 前 หน้า)
- 现在 (xiànzài): ปัจจุบัน (现 ปัจจุบัน + 在 อยู่)

### 14. หมวดสุขภาพและการแพทย์ (Health & Medical)
มักใช้โครงสร้างแบบขยาย บอกส่วนของร่างกายหรืออาการ:

- 医院 (yīyuàn): โรงพยาบาล (医 แพทย์ + 院 สถานที่)
- 感冒 (gǎnmào): หวัด/เจ็บหวัด (感 รู้สึก + 冒 ความเสี่ยง)
- 发烧 (fāshāo): ไข้ (发 ออก + 烧 ไฟ/ร้อน)
- 药品 (yàopǐn): ยา (药 ยา + 品 สิ่งของ)
- 疼痛 (téngtòng): ปวด (疼 เจ็บ + 痛 เจ็บ)
- 检查 (jiǎnchá): ตรวจสอบ (检 ตรวจ + 查 ตรวจสอบ)
- 治疗 (zhìliáo): รักษา (治 รักษา + 疗 รักษา)

### 15. หมวดครอบครัวและคนรู้จัก (Family & People)
มักใช้โครงสร้างแบบขยาย หรือคู่ขนาน:

- 亲戚 (qīnqi): ญาติ (亲 ใกล้ชิด + 戚 ญาติ)
- 邻居 (línjū): เพื่อนบ้าน (邻 ข้างเคียง + 居 อยู่)
- 朋友 (péngyǒu): เพื่อน (朋 เพื่อน + 友 เพื่อน)
- 同事 (tóngshì): เพื่อนร่วมงาน (同 ร่วม + 事 งาน)
- 客人 (kèrén): แขก (客 แขก + 人 คน)
- 家人 (jiārén): ครอบครัว (家 บ้าน + 人 คน)
- 同学 (tóngxué): เพื่อนร่วมชั้น (同 ร่วมกัน + 学 เรียน)

### 16. หมวดกีฬาและกิจกรรม (Sports & Activities)
มักใช้โครงสร้างแบบกริยา-กรรม:

- 游泳 (yóuyǒng): ว่ายน้ำ (游 ลอย + 泳 ว่าย)
- 跑步 (pǎobù): วิ่ง (跑 วิ่ง + 步 ก้าว)
- 跳舞 (tiàowǔ): เต้น (跳 กระโดด + 舞 การเต้น)
- 唱歌 (chànggē): ร้องเพลง (唱 ร้อง + 歌 เพลง)
- 运动 (yùndòng): ออกกำลังกาย (运 เคลื่อน + 动 ขยับ)
- 锻炼 (duànliàn): ออกกำลังกาย (锻 หล่อ + 炼 ฝึก)

### 17. หมวดเทคโนโลยีและสื่อสาร (Technology & Communication)
มักใช้โครงสร้างแบบขยาย:

- 网络 (wǎngluò): อินเทอร์เน็ต (网 ตาข่าย + 络 เครือข่าย)
- 邮件 (yóujiàn): อีเมล (邮 ไปรษณีย์ + 件 ชิ้น)
- 信息 (xìnxī): ข้อมูล (信 ข่าว + 息 ข่าว)
- 网站 (wǎngzhàn): เว็บไซต์ (网 ตาข่าย + 站 สถานี)
- 软件 (ruǎnjiàn): ซอฟต์แวร์ (软 อ่อน + 件 ชิ้น)
- 电话 (diànhuà): โทรศัพท์ (电 ไฟฟ้า + 话 คำพูด)
- 手机 (shǒujī): มือถือ (手 มือ + 机 เครื่องจักร)

### 18. หมวดเศรษฐกิจและธุรกิจ (Economy & Business)
มักใช้โครงสร้างแบบขยาย:

- 价格 (jiàgé): ราคา (价 ราคา + 格 รูปแบบ)
- 市场 (shìchǎng): ตลาด (市 ตลาด + 场 สถานที่)
- 公司 (gōngsī): บริษัท (公 สาธารณะ + 司 บริหาร)
- 银行 (yínháng): ธนาคาร (银 เงิน + 行 สถาบัน)
- 合同 (hétong): สัญญา (合 รวม + 同 ตกลง)
- 贸易 (màoyì): การค้า (贸 ค้า + 易 แลกเปลี่ยน)
- 利润 (lìrùn): กำไร (利 ผลประโยชน์ + 润 กำไร)

### 19. หมวดสิ่งแวดล้อมและสังคม (Environment & Society)
มักใช้โครงสร้างแบบขยาย:

- 环境 (huánjìng): สิ่งแวดล้อม (环 รอบ + 境 ขอบเขต)
- 社会 (shèhuì): สังคม (社 สังคม + 会 ชุมชน)
- 城市 (chéngshì): เมือง (城 เมือง + 市 ตลาด)
- 农村 (nóngcūn): ชนบท (农 การเกษตร + 村 หมู่บ้าน)
- 国家 (guójiā): ประเทศ (国 ประเทศ + 家 บ้าน)
- 社区 (shèqū): ชุมชน (社 สังคม + 区 เขต)

### 20. หมวดศิลปะและวัฒนธรรม (Arts & Culture)
มักใช้โครงสร้างแบบขยาย:

- 电影 (diànyǐng): ภาพยนตร์ (电 ไฟฟ้า + 影 เงา/ภาพ)
- 音乐 (yīnyuè): เพลง/ดนตรี (音 เสียง + 乐 ความสุข)
- 小说 (xiǎoshuō): นิยาย (小 เล็ก + 说 เล่า)
- 书法 (shūfǎ): ลายมือจีน (书 เขียน + 法 วิธี)
- 节日 (jiérì): เทศกาล (节 วันหยุด + 日 วัน)
- 艺术 (yìshù): ศิลปะ (艺 ฝีมือ + 术 เทคนิค)
